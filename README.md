# OPTLAB

OCR前处理工具箱，基于Qt和Leptonica。

## 功能

- [x] 图片反色
- [x] 旋转
- [x] 缩放
- [x] 二值化
- [x] 去倾斜 
- [x] 去噪
- [x] 膨胀
- [x] 腐蚀
- [ ] 扫描文档边缘识别
- [ ] 命令行模式
